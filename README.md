# eMenu

Restaurant menu system made for reqruitment purposes

### Prerequisites

Virtual environment with Python 3.6+

 
### Installing

* Clone repository
* Install needed libraries:

- geopy
- flask
- pyyaml
- requests

- create config.yaml file in project's root directory

It must contain values as below

google_api_key : "your google geocoding api key"
google_js_api_key : "your google maps js api key"
aqicn_api_key: "your aqi cn api key"


* Run development server:

python flask_app.py

* Visit 127.0.0.1:5000
